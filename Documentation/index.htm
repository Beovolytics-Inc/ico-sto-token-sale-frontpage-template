Installation

If you don't want to install theme yourself you can always order our WordPress and Theme installation service.

You can skip few basic installation steps in this guide
if you watch and follow our Installation video guide below:

If you don't watched video follow manual steps:

Unzip the file that you download after purchase. The following files and folders will be in the unzipped file:

  • cryptibit.zip (Packed theme file for install)
  • cryptibit folder - theme for manual FTP upload
  • Documentation/index.html (This file)
  • Documentation/release-history.html - Update instructions for theme updates

The theme can be uploaded in two ways:

  • FTP upload: Using your FTP program, upload the non-zipped cryptibit folder into the /wp-content/themes/ folder on your server.
  • WordPress upload: Navigate to Appearance > Install Themes > Upload. Go to browse, and select the zipped cryptibit.zip file. Hit Install Now and the theme will be uploaded and installed.

Once the theme is uploaded, you need to activate it. Go to Appearance > Themes and activate CryptiBIT theme.

If you need help installing WordPress see the link here : http://codex.wordpress.org/Installing_WordPress

After the install finish, go to Theme Control Panel page, then click Save button to import default settings values to database. You don't need to do this if you use 1-Click Demo import feature.

MANUAL DEMO DATA IMPORT (NOT RECOMMENDED):

We does not recommend to use manual XML import. Use this only if you can't import demo data correctly with 1-Click demo data import for some reason (does not related to hosting PHP.ini limits settings - manual import require much higher PHP.ini limits).

  1. Go to Tools > Import > WordPress. If you don't installed WP Importer plugin before you will be asked to install it.
  2. Select content.xml demo file from /wp-content/plugins/cryptibit-theme-addons/inc/oneclick-demo-import/demo-files/1/ folder (you need to download this folder files from your FTP first, after theme plugins installation).
  3. Submit Upload file and import.
  4. Assign posts author to an existing user account.
  5. You can check to Download and import file attachments checkbox to import all sample images.

    Please not that this option require good server with a lot of server memory and CPU available. If you have cheap shared hosting don't check this option. Importing all images will take 10 minutes or more (WordPress will download & generate around 30 MB of test images), depending on your server hardware.

    We recommend you to import sample images on local test servers without low memory/time limitation on your PC/Mac (We recommend to use MAMP). You need to set large values for PHP time limit (1 hour or more) and PHP memory limit (256MB or more) in php.ini configuration file on your server before you start importing images.
  6. Submit and wait when WP tools download data and attachments to your WordPress site.
  7. If you don't see Success message after import or your page just stop loading with or without any error, and your site does not look like our demo this mean your server configuration does not have enougth scripts time limit and memory limit to import all sample data. You should contact your hosting/server provider and ask to increase PHP memory limit and time limits.

    Check this article how to solve demo data import problems.

    Please note that you does not have legal rights to use any photos and images imported with manual XML import, this images used for demo purposes only. You must change demo images to your own real images.

Additionally you can import sample widgets (sidebars) from our demo:

  1. Go to Appearance > Widgets and delete all default WordPress widgets from all theme sidebars.
  2. Install and activate Widget Importer & Exporter plugin.
  3. Go to Tools > Widget Importer & Exporter.
  4. Click Browse... button and select widgets.json file from /wp-content/plugins/cryptibit-theme-addons/inc/oneclick-demo-import/demo-files/1/ folder (you need to download this folder files from your FTP first, after theme plugins installation).
  5. Click Import Widgets button.
  6. Now you can go to Appearance > Widgets and reconfigure sample widgets in all sidebars to your purposes.
If you don't imported dummy data you must create theme pages and menus manually.

SPECIAL CSS CLASS NAMES FOR MENU ITEMS

You can use special predefined CSS class names for main manu items to change its look.

You can use this special CSS classes:

  1. pull-right - use this to move your element to right side of main menu (this class work only for menu below header).
  2. mgt-animated - use this to add hover animation for menu element (this class work only for menu below header).

Child theme usage

A child theme is a special kind of WordPress theme that inherits all of the design elements of its parent theme. A child theme with no customization is identical to its parent.

WHEN YOU DON'T NEED TO USE CHILD THEME:

1) Change theme options.
2) Add/change plugins.
3) Change/add CSS styles to theme (use Customize > Additional CSS instead).
4) Change/add JavaScript in theme (use Theme Control Panel > Custom JS instead).
5) Add new features to theme without code (use Plugins instead).

WHEN YOU NEED TO USE CHILD THEME:

1) You are WordPress developer and want to code your PHP functions to extend theme features or add your own.
2) You are WordPress developer and want to override theme PHP functions with your own functions to change something.
3) You want to completely change HTML templates in theme or add your own templates (you need to know PHP/HTML coding for this).

HOW TO INSTALL & USE CHILD THEME:

1) Install parent theme first, activate it, install and activate all theme plugins first.
2) Install child theme in the same way from ZIP file and activate it.
3) Refer to WordPress documentation how to work with child theme: https://codex.wordpress.org/Child_Themes

You can find Child Theme in "Child theme" folder inside full theme ZIP archive (All files&Documentation) from Themeforest. To install child theme you need to upload it to WordPress as regular theme.
