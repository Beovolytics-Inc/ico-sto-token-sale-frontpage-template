Installation

You must have WPBakery Page Builder plugin installed to use our plugin. Our plugin add Timeline/Roadmap element to WPBakery Page Builder elements list.

Unzip ZIP file that you download after purchase on CodeCanyon. The following files and folders will be in the unzipped file:

  • mgroadmaptimeline.zip (Packed plugin file for install)
  • mgroadmaptimeline folder - plugin for manual FTP upload
  • Documentation (This file)

Plugin can be uploaded in two ways:

  • FTP upload: Using your FTP program, upload the non-zipped mgroadmaptimeline folder into the /wp-content/plugins/ folder on your server.
  • WordPress upload: Navigate to Plugins > Add New > Upload plugin. Go to browse, and select the zipped mgroadmaptimeline.zip file. Hit Install Now and the plugin will be uploaded and installed.

Once the plugin is uploaded, you need to install and activate it. Go to Plugins and activate Ultimate RoadMap Timeline plugin.

If you need detailed help installing Wordpress plugin see the link here : http://codex.wordpress.org/Managing_Plugins

CREATING TIMELINE/ROADMAP ELEMENTS

You can create unlimited timelines with unlimited elements everywhere on your content area on site (pages, posts, custom post types, etc).

  1. Go to page where you want to create your timeline and enable backend or frontend WPBakery Page Builder edit mode (we will use backend mode as example in this Documentation).
  2. Click + to add new page builder element and select MGT Timeline element from list:




  3. You will see Timeline element options dialog where you can configure all element options for your timeline event - check next documentation section for available options help.
  4. Set options that you like and click Save changes.
  5. Add all other timeline elements and Publish your page.
  6. You can add Timeline elements inside any rows or columns, but make sure your column have enough width to fit timeline elements nicely.

Timeline/ROADMAP element options

  1. Timeline item title - your event title (for ex. "Store opening").
  2. Title color - text color for your title if you want to override default color.
  3. Timeline item text - your event description. You can use HTML and third party shortcodes here, insert images or videos. If you want to add video use its embeding code (for ex. for Youtube) in Text mode for content editor.
  4. Timeline image - image for your event. Use if you want to add image after text or instead of text.
  5. Timeline date year - Year for your event date. You can add full date or any text here to display it with large text.
  6. Timeline date month - Month for your event date. You can add any text here (for ex. "Q1-Q4") or leave it empty.
  7. Year text color - text color for your year if you want to override default color.
  8. Month text color - text color for your month if you want to override default color.
  9. Text block style - change background appearance for your timeline item text element.
  10. Text block border style - change corners style for your timeline item text element.
  11. Text color - text color for your timeline item text element if you want to override default color.
  12. Text background color - text background color for your timeline item text element if you want to override default color. If you selected bordered or double bordered text block style (#9) property text background color will change border color (not background).
  13. Timeline vertical line position - This is important option for your timeline layout. If you adding first (top) timeline element you must choose "Bottom" line position here. If you adding last (bottom) timeline element here you must choose "Top" line position here. If you adding middle element - use "Both" option value here.
  14. Date position - you should alternate this value for your timeline elements to display it at left, right, left, right, etc. So for example your odd elements will have "Left" here, your even elements will have "Right" here.
  15. Timeline vertical line color - vertical line color for this timeline element if you want to override default color.
  16. Pointer style - style for your element pointer (line from element text block to vertical line).
  17. Timeline element link url - add url to your element title and icon if you wish.
  18. Open link in new tab - used for Timeline element url link.
  19. Disable icon - use this option to disable icon and show just small bullet for this element in vertical line.
  20. Timeline icon library and icon - choose icon that you want to use from available. This option will be ignored if you disabled icon.
  21. Icon size - Change icon size (note: this does not change icon background size). You does not need to input "px" here, input just integer value, for ex. 32. This option will be ignored if you disabled icon.
  22. Icon color - icon color for your month if you want to override default color. This option will be ignored if you disabled icon.
  23. Icon background color - change icon background color. This option will work even if you disabled icon (you will change small bullet background color).
  24. Timeline Icon Border style - change border style for icon box. This option will work even if you disabled icon (you will change small bullet border style).
  25. Add mouse hover shadow effect - use to add mouse hover animation for icon or bullet.
  26. CSS Animation - on scroll appear animation for your timeline element.

TROUBLESHOOTING

This plugin will work fine with any well-coded WordPress themes that works well with WPBakery Page Builder plugin. If you see missing or broken styles, incorrect layouts for Timeline elements this means your theme override our plugin styles in incorrect way and you need to contact theme developer for help (he need to modify theme styles to not override our plugin styles).

Translations

This plugin is translation ready and WPML plugin compatible and this mean you can easy adapt it to your language or use it multilanguage. You can translate plugin manualy with WPML or LocoTranslate plugins.

Manual theme translation

We included POT files for translation with theme:

  • /wp-plugins/mgroadmaptimeline/languages/ - main plugin translations

Read this section on Wordpress website about WP translations.

Translation with LOCO TRANSLATE PLUGIN

If you installed and activated LocoTranslate Translation Manager you can translate plugin with it:

  1. Install and activate LocoTranslate
  2. Go to Loco Translate > Manage translations
  3. Click New language near our plugin in list
  4. Translate all plugin strings in admin panel
